# vnshop 商城
## 项目架构
前后分离

前端：
    使用vue cli 脚手架方式搭建

api：
    使用node express框架搭建

## 项目目录
cd vnshop
mkdir -p client note server

vnshop
    client #客户端代码，里面放的是vue项目工程文件

    note #搭建项目的详细笔记
        
    server #node express 代码
